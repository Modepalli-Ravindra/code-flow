**CodeFlow** is a personal, step-by-step coding learning engine designed to help complete beginners bridge the gap between "writing code" and "understanding logic." 

Unlike traditional IDEs, CodeFlow focuses on the **Universal Concept Model**, mapping different languages (JavaScript, Python, Java, SQL) into core building blocks like Variable Creation, Loop Iteration, and the Recursion Stack.

   ```bash
   git clone https://github.com/Modepalli-Ravindra/code-flow.git
   cd code-flow
   ```

2. **Install Dependencies**:
   ```bash
   # Install backend deps
   cd server
   npm install

   cd ..
   npm install
   ```

3. **Run the Engines**:
   ```bash
   # Start the backend (terminal 1)
   npm run server
   
   # Start the frontend (terminal 2)
   npm run dev
   ```

4. **Learn!**: Open `http://localhost:5173` and start your first "Time-Travel" coding session.
